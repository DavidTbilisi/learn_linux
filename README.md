## ­ЪљД Linux-рЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў

### ­ЪЊЂ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

* `ls`, `cd`, `pwd`, `mkdir`, `rm`, `cp`, `mv`
* рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ, рЃгрЃљрЃерЃџрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃљ
* рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљ([UnixArena][1])

### ­ЪЊЂ `ls` РђЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃўрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃљрЃю рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
ls            # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА
ls -l         # рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃќрЃћ
ls -a         # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃарЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃф (рЃЊрЃљрЃгрЃДрЃћрЃЉрЃБрЃџрЃў .-рЃўрЃЌ)
```

---

### ­ЪЊЂ `cd` РђЊ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
cd /etc       # рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА /etc рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў
cd ~          # рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа рЃАрЃљрЃгрЃДрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў
cd ..         # рЃћрЃарЃЌрЃў рЃЊрЃЮрЃюрЃўрЃЌ рЃќрЃћрЃЏрЃЮрЃЌ
```

---

### ­ЪЊЂ `pwd` РђЊ рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃАрЃарЃБрЃџрЃў рЃњрЃќрЃљрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
pwd           # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА, рЃАрЃљрЃЊ рЃўрЃЏрЃДрЃЮрЃцрЃћрЃЉрЃўрЃЌ рЃцрЃљрЃўрЃџрЃБрЃа рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў
```

---

### ­ЪЊЂ `mkdir` РђЊ рЃљрЃ«рЃљрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃЦрЃЏрЃюрЃўрЃА рЃљрЃ«рЃљрЃџ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
mkdir projects          # рЃЦрЃЏрЃюрЃўрЃА "projects" рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА
mkdir -p dir1/dir2      # рЃЦрЃЏрЃюрЃўрЃА рЃЕрЃљрЃЊрЃњрЃЏрЃБрЃџ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃА
```

---

### ­ЪЊЂ `rm` РђЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃю рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃгрЃљрЃерЃџрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃерЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃљрЃю рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
rm file.txt             # рЃерЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃА
rm -r folder/           # рЃерЃџрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА рЃЊрЃљ рЃЏрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА
rm -rf folder/          # рЃФрЃљрЃџрЃўрЃЌ рЃерЃџрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА (рЃАрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃЮ!)
```

---

### ­ЪЊЂ `cp` РђЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА/рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃА рЃљрЃю рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
cp file.txt backup.txt          # рЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃА
cp -r dir1/ dir2/               # рЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ
```

---

### ­ЪЊЂ `mv` РђЊ рЃцрЃљрЃўрЃџрЃўрЃА/рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ рЃљрЃю рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃА рЃљрЃю рЃљрЃФрЃџрЃћрЃЋрЃА рЃљрЃ«рЃљрЃџ рЃАрЃљрЃ«рЃћрЃџрЃА рЃцрЃљрЃўрЃџрЃА/рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
mv old.txt new.txt              # рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ
mv file.txt /tmp/               # рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ рЃАрЃ«рЃЋрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў
```

---

рЃњрЃўрЃюрЃЊрЃљ рЃњрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃќрЃћ (рЃЏрЃљрЃњ. `chmod`, `man`, `grep`)?


### ­ЪДЉРђЇ­Ъњ╗ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ

* `nano`, `vim`, `cat`, `less`, `more`
* рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ([UnixArena][1], [Proxmox VE][2])

Sure, here's the same format for `nano`, `vim`, `cat`, `less`, and `more`, in Georgian, Markdown-ready:

### ­ЪЊЮ `nano` РђЊ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
nano file.txt          # рЃ«рЃАрЃюрЃўрЃА рЃљрЃю рЃЦрЃЏрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
```

> рЃњрЃљрЃЏрЃЮрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ рЃЊрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ: `Ctrl + O`, рЃерЃћрЃЏрЃЊрЃћрЃњ `Ctrl + X`

---

### ­ЪЊЮ `vim` РђЊ рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃФрЃџрЃўрЃћрЃарЃў рЃЊрЃљ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў CLI-рЃерЃў.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
vim file.txt           # рЃ«рЃАрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
```

> рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃарЃћрЃЪрЃўрЃЏрЃћрЃЉрЃў:

* **i** РђЊ рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃў рЃарЃћрЃЪрЃўрЃЏрЃў
* **Esc** РђЊ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃў
* **:wq** РђЊ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃАрЃЋрЃџрЃљ
* **:q!** РђЊ рЃњрЃљрЃЏрЃЮрЃАрЃЋрЃџрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ

---

### ­ЪЊќ `cat` РђЊ рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
cat file.txt           # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА
cat file1 file2        # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЮрЃарЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃўрЃЌ
```

---

### ­ЪЊќ `less` РђЊ рЃЊрЃўрЃЊрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃљрЃЊ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃА рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃљрЃЊ, рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃЊрЃўрЃЊрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
less file.txt          # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА рЃАрЃЦрЃарЃЮрЃџрЃўрЃарЃћрЃЉрЃљрЃЊ рЃарЃћрЃЪрЃўрЃЏрЃерЃў
```

> рЃњрЃљрЃЏрЃЮрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ рЃЊрЃљрЃљрЃГрЃўрЃарЃћ `q`

---

### ­ЪЊќ `more` РђЊ рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ рЃњрЃЋрЃћрЃарЃЊрЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА рЃћрЃарЃЌ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ, рЃћрЃбрЃљрЃърЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
more file.txt          # рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃА рЃћрЃбрЃљрЃърЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ
```

> рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ рЃњрЃљрЃЊрЃљрЃ«рЃЋрЃћрЃЋрЃљ, `q` рЃњрЃљрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ

---

### ­Ъћљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃљ

* `chmod`, `chown`, `ls -l`
* рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ([UnixArena][1])

Here you go Рђћ documentation for `chmod`, `grep`, and `find` in the same style:

---

### ­Ъћљ `chmod` РђЊ рЃцрЃљрЃўрЃџрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃфрЃЋрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃю рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃА.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
chmod 755 script.sh         # рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃА рЃАрЃарЃБрЃџрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ, рЃАрЃ«рЃЋрЃћрЃЉрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ/рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
chmod +x script.sh          # рЃљрЃФрЃџрЃћрЃЋрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃљрЃА
chmod -R 644 *.txt          # рЃфрЃЋрЃџрЃўрЃА .txt рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃА рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ
```

---

### ­ЪћЇ `grep` РђЊ рЃбрЃћрЃЦрЃАрЃбрЃерЃў рЃФрЃўрЃћрЃЉрЃљ

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃћрЃФрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃА рЃцрЃљрЃўрЃџрЃерЃў рЃљрЃю рЃерЃарЃћрЃЏрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
grep "error" logfile.txt        # рЃћрЃФрЃћрЃЉрЃА рЃАрЃўрЃбрЃДрЃЋрЃљ "error"-рЃА рЃцрЃљрЃўрЃџрЃерЃў
grep -i "hello" file.txt        # рЃўрЃњрЃюрЃЮрЃарЃўрЃарЃћрЃЉрЃА рЃърЃљрЃбрЃљрЃарЃљ/рЃЊрЃўрЃЊ рЃљрЃАрЃЮрЃћрЃЉрЃА
grep -r "TODO" ./project        # рЃћрЃФрЃћрЃЉрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃарЃћрЃЎрЃБрЃарЃАрЃўрЃБрЃџрЃљрЃЊ
```

---

### ­ЪДГ `find` РђЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃФрЃўрЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃБрЃа рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў

**рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ:** рЃърЃЮрЃБрЃџрЃЮрЃЉрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃерЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЎрЃарЃўрЃбрЃћрЃарЃўрЃБрЃЏрЃўрЃЌ.

**рЃЏрЃЮрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:**

```bash
find . -name "*.log"            # рЃћрЃФрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ .log рЃцрЃљрЃўрЃџрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃЊрЃљрЃю
find /etc -type f -size +1M     # рЃърЃЮрЃБрЃџрЃЮрЃЉрЃА 1MB-рЃќрЃћ рЃЊрЃўрЃЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА
find . -mtime -1                # рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЉрЃЮрЃџрЃЮ 1 рЃЊрЃдрЃћрЃерЃў рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ
```

---


### ­ЪД░ рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

* `man`, `which`, `whereis`, `locate`, `find`
* рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃФрЃўрЃћрЃЉрЃљ([Reddit][3], [UnixArena][1])

### ­ЪДф рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

* `ps`, `top`, `htop`, `kill`, `nice`, `renice`
* рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЋрЃљ

### ­Ъїљ рЃЦрЃАрЃћрЃџрЃБрЃарЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

* `ping`, `curl`, `wget`, `netstat`, `ss`, `traceroute`
* рЃЦрЃАрЃћрЃџрЃБрЃарЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃЊрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃўрЃљрЃњрЃюрЃЮрЃАрЃбрЃўрЃЎрЃљ([roadmap.sh][4])

### ­ЪЊд рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

* `apt`, `yum`, `dnf`, `pacman`, `snap`
* рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ, рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ

### ­ЪДЉРђЇ­ЪћД рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃ»рЃњрЃБрЃцрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

* `useradd`, `usermod`, `userdel`, `groupadd`, `passwd`
* рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃ»рЃњрЃБрЃцрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЋрЃљ([UnixArena][1])

### ­ЪќЦ№ИЈ рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў

* `systemctl`, `service`, `journalctl`
* рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ, рЃњрЃљрЃЕрЃћрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ([Proxmox VE][2])

### ­ЪДЙ рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ

* `dmesg`, `uptime`, `free`, `df`, `du`
* рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЊрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў
